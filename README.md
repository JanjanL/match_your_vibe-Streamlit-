# Match your vibe

The app "Match your vibe" is a demonstration of deploying an interactive dashboard and the function from Streamlit for students.

By using a dataset of spotify which includes audio features (energy, valence, danceability) and artist names, a two-column app was deployed with ainteractive quiz on the left and resulting in the best fit for the user's vibe on the right.
(Dataset: https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset)

What you’d get:
Top 3 best matches your vibe, "Wow this track is basically made for your vibe — Dosai Amma Dosai Food Song. Match level: XX.

Behind the scenes: compute quiz → target feature vector → rank artists by similarity.
